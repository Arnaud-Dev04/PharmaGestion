# SYSTÈME DE GESTION DE PHARMACIE
Version 1.0

## Installation

1. Décompressez le dossier "PharmaGestion" où vous voulez
2. Double-cliquez sur "Lancer_PharmaGestion.bat"
3. Le navigateur s'ouvre automatiquement
4. Connectez-vous avec vos identifiants

## Identifiants par défaut

**Administrateur:**
- Nom d'utilisateur: admin
- Mot de passe: admin

**Pharmacien:**
- Nom d'utilisateur: pharmacien
- Mot de passe: pharmacien

⚠️ CHANGEZ CES MOTS DE PASSE après la première connexion !

## Utilisation

- L'application fonctionne 100% hors ligne
- Toutes les données sont sauvegardées dans "database.db"
- Pour fermer l'application, fermez la fenêtre noire du terminal

## Sauvegarde

Pour sauvegarder vos données:
1. Fermez l'application
2. Copiez le fichier "database.db"
3. Conservez-le en lieu sûr

Pour restaurer:
1. Remplacez "database.db" par votre sauvegarde

## Support

En cas de problème:
- Vérifiez que le port 8000 n'est pas utilisé
- Redémarrez l'application
- Vérifiez vos permissions Windows

## Configuration Requise

- Windows 7/8/10/11
- 100 MB d'espace disque
- Connexion Internet: NON requise
