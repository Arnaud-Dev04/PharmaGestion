# Pharmac+ Frontend

Frontend React + Vite pour le système de gestion de pharmacie Pharmac+.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera accessible sur `http://localhost:5173`

## Build Production

```bash
npm run build
```

## Stack Technique

- React 18
- Vite
- TailwindCSS
- React Router DOM
- React Query
- Axios
- Zustand
- Lucide React
- React Hook Form + Zod

## Structure

```
src/
├── components/      # Composants réutilisables
├── pages/          # Pages de l'application
├── services/       # Services API
├── context/        # Context providers
├── hooks/          # Custom hooks
└── utils/          # Utilitaires
```

## Configuration

Le backend doit tourner sur `http://localhost:8000`
